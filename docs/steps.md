## Run Docker

Build docker by directly running the `docker-compose up -d --build` in the terminal or by running the `start.sh` file, first time build take a lot of time. 

```bash
./start.sh
```

To check status of docker

```bash
docker-compose ps
```

To stop docker

```bash
docker-compose down
```
